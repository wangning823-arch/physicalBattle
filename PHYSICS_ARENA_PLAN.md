# 物理竞技场 - 三国杀物理版

## 1. 游戏概念

**核心玩法**: 2名玩家站在矩形物理竞技场内，通过打出手中的"物理定律卡牌"利用真实物理规则将对手推出场地，最后留在场地内的玩家获胜。

**目标用户**: 桌游爱好者、物理学习者、派对游戏玩家

**视角**: 侧视角（2D横版），有重力影响

**平台**: Web端桌面游戏（可扩展至移动端）

---

## 2. 核心机制设计

### 2.1 场地物理参数
```
竞技场参数:
- 形状: 圆形平台（圆柱形容器的俯视图）
- 半径: 8米
- 重力: g = 9.8 m/s² (向下)
- 地面摩擦系数: μ = 0.3
- 边界: 圆形平台边缘 → 下方悬崖 (掉出即出局)
- 侧面: 圆形内壁，玩家可贴壁站立
- 空气阻力: 轻微阻尼
```

### 2.2 玩家物理参数
```
玩家状态:
- 质量: 70kg (默认)
- 尺寸: 1m × 1m 方形
- 与圆形内壁的摩擦力: Ff = μ × N
- 最大速度限制: 15 m/s
- 初始位置: 圆形平台中心两侧，距边缘 2m

### 2.3 卡牌系统设计

#### 卡牌分类

| 卡牌类型 | 示例卡牌 | 物理原理 | 效果描述 |
|---------|---------|---------|---------|
| **动量类** | 动量冲击 | 动量守恒 P=mv | 对目标施加冲量，使其获得速度 |
| **力学类** | 万有引力 | F=Gm₁m₂/r² | 在场地内产生引力场 |
| **能量类** | 弹性弹射 | 动能定理 | 将自身弹性反弹 |
| **力学类** | 超级推力 | 牛顿第二定律 F=ma | 对自身/对手施加巨大加速度 |
| **摩擦类** | 润滑油 | 改变摩擦系数 | 降低某区域摩擦力 |
| **碰撞类** | 完全弹性碰撞 | 动量+能量守恒 | 完美反弹碰撞 |
| **冲量类** | 延时炸弹 | 脉冲式力 | 延时触发的大冲量 |
| **反作用类** | 喷气加速 | 动量守恒 | 获得反向推力 |

### 2.4 能量系统
```
能量机制:
- 每回合开始: +3 能量点
- 最大能量: 10 点
- 卡牌消耗: 1-3 能耗不等

### 2.5 游戏流程
```
回合结构:
1. 能量恢复阶段
   - 所有玩家恢复 +3 能量

2. 抽卡阶段
   - 从公共牌堆抽取 2 张卡

3. 行动阶段
   - 玩家按顺序行动
   - 每个玩家可以:
     a) 出1张卡牌 (主动攻击/防御/位移)
     b) 不出牌

4. 物理结算阶段
   - 计算所有力、速度、位移
   - 检测碰撞
   - 检测出界 (离开圆形场地 = 淘汰)
   - 摩擦力结算

5. 回合结束
   - 地形卡效果减少一回合
```

### 4.2 力的施加接口

```javascript
// 力施加标准接口
function applyForce(body, force) {
  const mass = body.mass;
  const acceleration = {
    x: force.x / mass,
    y: force.y / mass
  };
  Body.setAcceleration(body, acceleration);
}

// 冲量施加
function applyImpulse(body, impulse) {
  const velocity = {
    x: body.velocity.x + impulse.x / body.mass,
    y: body.velocity.y + impulse.y / body.mass
  };
  Body.setVelocity(body, velocity);
}

// 摩擦力计算
function calculateFriction(body, frictionCoeff) {
  const normalForce = body.mass * 9.8;
  const frictionForce = frictionCoeff * normalForce;
  return frictionForce;
}
```

## 7. 物理公式参考

### 7.1 核心公式

```
动量守恒:        Σp₁ = Σp₂ = const
冲量:            J = F·Δt = Δp = m·Δv
动能:            Eₖ = ½mv²
摩擦力:          Ff = μ·N
万有引力:        F = G·m₁·m₂/r²
完全弹性碰撞:    e = 1 (恢复系数)
非弹性碰撞:      0 < e < 1
```

### 7.2 卡牌数值平衡

```
卡牌效果数值 (基于70kg玩家、μ=0.3):
- 动量冲击:      500 kg·m/s → Δv ≈ 7.1 m/s
- 万有引力:      F = 100N → a ≈ 1.4 m/s²
- 喷气背包:      v = 5 m/s
- 质量改变:      ×2 或 ×0.5
```
